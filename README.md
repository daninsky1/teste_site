Criar e Iniciar Python venv:
<br/>
Windows:<br/>
`py -3 -m venv venv`<br/>
`.\venv\scripts\activate.bat`<br/>
Linux:<br/>
`python3 -m venv venv`<br/>
`./venv/bin/activate`<br/><br/>

Instalar e Iniciar Flask:<br/>
`pip install Flask`<br/>
`flask run`
